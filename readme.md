# Installation Guide

## Project Structure

The root directory contains two folders:

* **frontend/**
* **backend/**

---

## Frontend Setup

1. Navigate to the frontend directory:

   ```bash
   cd frontend
   ```
2. Install dependencies:

   ```bash
   npm i
   ```
3. Start the frontend server:

   ```bash
   npm start
   ```

---

## Backend Setup

1. Navigate to the backend directory:

   ```bash
   cd backend
   ```
2. Install dependencies:

   ```bash
   npm i
   ```
3. Start the backend server:

   ```bash
   node server.js
   ```

---

Both servers should now be running successfully.
